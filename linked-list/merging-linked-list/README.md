### Merge a linked list into another linked list at alternate positions
---

#### Before merge
**Example 1**:
1st linked list:
```5 -> 7 -> 17 -> 13 -> 11```

2nd linked list:
```12 -> 10 -> 2 -> 4 -> 6```

**Example 2**:
1st linked list:
```1 -> 2 -> 3```

2nd linked list:
```4 -> 5 -> 6 -> 7 -> 8```

---

#### After merge
**Example 1**:
1st linked list:
```5 -> 12 -> 7 -> 10 -> 17 -> 2 -> 13 -> 4 -> 11 -> 6```

2nd linked list:
```<EMPTY>```

**Example 2**:
1st linked list:
```1 -> 4 -> 2 -> 5 -> 3 -> 6```

2nd linked list:
```7 -> 8```
